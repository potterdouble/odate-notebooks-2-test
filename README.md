# Databases

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/ChantalMB/odate-notebooks-wip/databases)

These notebooks demonstrate how to ingest a variety of csv (or other format) files into a single SQL database. They show how to query a database, and how to push the results of a query into a dataframe for further analysis or visualization.
