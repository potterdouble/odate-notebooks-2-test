# Agent Based Models

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/ChantalMB/odate-notebooks-wip/abm)

> **Note:** This Binder may take ~5min to launch if it hasn't bee built recently, due to the system packages required.

Many of these notebooks use the [Project Mesa](https://github.com/projectmesa/mesa) 'mesa' package for building and running agent based models. The sample notebooks are in the relevant model folders. They show how to build and run parameter sweeps. 

Start with the Forest Fire model - it is one of the best known introductory models in the field. As you experiment with these models, ask yourself, 'what would it take for this to be a model of an archaeological concept?'.
